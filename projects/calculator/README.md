# C Calculator Project

A simple, interactive command-line calculator built in C.

## Features
- **Arithmetic Operations**: Addition, Subtraction, Multiplication, Division.
- **Advanced Operations**: Modulus (Remainder).
- **Error Handling**: Prevents division by zero and handles invalid inputs.
- **Looping**: Runs continuously until you choose to exit.

## How to Build and Run

### Prerequisites
You need a C compiler like GCC installed.

### Compilation
Open your terminal/command prompt to this folder and run:

```bash
gcc calculator.c -o calculator
```

### Execution
Run the generated executable:

**Windows:**
```bash
./calculator.exe
```

**Linux/Mac:**
```bash
./calculator
```

## Usage Guide
1. Run the program.
2. Select an operation number from the menu (1-6).
3. Input two numbers when prompted.
4. View the result!
5. Select '6' to exit.
