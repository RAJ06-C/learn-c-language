# Advanced Concepts Examples

This folder containing examples of advanced C programming concepts.

## Files

| File | Description |
|------|-------------|
| `01_function_pointers.c` | Storing function addresses, implementing callbacks, and lookup tables. |
| `02_variadic_functions.c` | Creating functions like `printf` that take indefinite arguments. |
| `03_bit_fields_unions.c` | Memory optimization with bit fields and type viewing with Unions. |
| `04_error_handling.c` | Simulating exceptions with `setjmp` and `longjmp`. |

## How to Compile & Run

```bash
gcc 01_function_pointers.c -o func_ptr
./func_ptr
```
