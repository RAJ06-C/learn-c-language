# Preprocessor Directives Examples

This folder demonstrates the capabilities of the C Preprocessor.

## Files

| File | Description |
|------|-------------|
| `01_macros.c` | Basics of `#define` for constants and function-like macros. |
| `02_conditional_compilation.c` | Using `#ifdef`, `#else`, `#endif` to control what gets compiled. |
| `03_macro_operators.c` | Advanced operators: Stringify (`#`) and Token Pasting (`##`). |
| `04_predefined_macros.c` | Using standard macros like `__DATE__`, `__FILE__`, `__LINE__`. |

## How to Compile & Run

```bash
gcc 01_macros.c -o macros
./macros
```
